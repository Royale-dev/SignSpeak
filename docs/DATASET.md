SignSpeak Dataset v1
====================

Language:
- American Sign Language (ASL)

Supported Classes:
- A
- B
- C
...
- Y

Excluded:
- J
- Z

Samples per class:
- 300

Total samples:
- 7200

Input:
- 21 hand landmarks

Features:
- x
- y
- z

Output:
- One label

CSV format:
label,x0,y0,z0,...

Collection rules:
- Different distances
- Different positions
- Slight wrist rotation
- Different lighting
- Natural variation
